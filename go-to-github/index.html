Source on GitHub

We use the Git distributed revision control system to archive Temoa's source code. The source code and data can be accessed through the link below.

Temoa Source Code

Since the concept of revision control may be new to many of the Temoa Project's audience, an introduction may be in order. In our GitHub repository, you can view both the Temoa code as well as updates made to the code over time. For information on how to clone the repository, see the Download page. If you have any questions, please make use of our mailing list.
