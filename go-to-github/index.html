Source on GitHub

We have opted to use the Git distributed revision control system for Temoa's logic. There are many publically available services that freely host open source project code. We chose GitHub.

Thus, please be aware that the links at the bottom of this page leave the Temoa Project website. We do not operate these alternate sites and are not responsible for any transaction performed on or through them.

The disclaimer aside, the concept of revision control may be new to many of the Temoa Project's audience, so an introduction may be in order. If you have any questions, please make use of our mailing list.
