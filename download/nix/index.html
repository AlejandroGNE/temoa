Installing Temoa on Linux

The number of Nix distributions is fairly large, so we present the general idea. Please adjust as appopriate for your installation. As ever, if you need help, please ask on the Temoa Forum. The gist is:

  • Ensure you are running Python v2.7.
  • Install Subversion (for download Coopr).
  • Install the Coopr "Version of the Day".
  • Install the GLPK solver.
  • Download and use the Temoa EEO model.
  • (Optional) Download the example data files for testing.

Instructions for Debian/Ubuntu

Open a terminal (often Ctrl+Alt+T will work). In the terminal type (or better: copy and paste) the commands after the prompt symbols ($):

  # go to you home directory (usually /home/username), or wherever
  # you want to install Coopr
$ cd

  # update your system's index of available packages
$ sudo apt-get update

  # install three programs: glpsol (a basic LP solver), subversion,
  # and graphviz (for auto-generated graphs from Temoa)
$ sudo apt-get install  glpk-utils  subversion  graphviz

$ python -V
Python 2.7.4                # Ensure this reads like: Python 2.7.x

  # the --trunk option means you'll be able to update much easier later.
$ python coopr_votd --trunk

  # Add the coopr binaries to your path.  This will enable you to just
  # type 'coopr_python' to execute Coopr.  Note the double caret '>'.
  # If you only use a single caret, then you will overwrite the file
  # '.profile' rather than append to it.  It is probably best to just
  # copy and paste this line.
$ echo "PATH=\"\$PATH\":$(pwd)/coopr/bin" >> .profile

Logout and then back in, then run $ pyomo to test the Coopr installation.

You may execute the Temoa EEO model as exemplified on the main download page.

Instructions for Fedora

Open a terminal and replace the apt-get commands from above with yum
