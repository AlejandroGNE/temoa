Installing Temoa on Linux

The number if Nix distributions is fairly large, so we present here only two distribution specific instructions. The gist is:

  • Ensure you are running Python v2.7 or at least have Python v2.7 on the path.
  • Ensure you have Python SetupTools installed.
  • Use SetupTools to install Coopr.
  • Install the GLPK solver.
  • Download and use the Temoa EEO model.
  • (Optional) Download the example data files for testing.

Instructions for Debian/Ubuntu

  1. Open the your Ubuntu Software Center, and search for Python2.7. Make sure it is installed.
  2. Install the package python-setuptools.
  3. Install the package glpk.
  4. From a command line, execute sudo easy_install-2.7 -U coopr.

Once the last completes, you may execute the Temoa EEO model as exemplified on the main download page.

Instructions for Fedora

Open a terminal, and type (or copy/paste) this series of commands:

  1. sudo yum install python2.7 python-setuptools glpk
  2. sudo easy_install-2.7 -U coopr

Similar to the Debian-based instructions, assuming all goes well, you may test your setup as indicated on the main download page.
