Download

The abstract formulation of the Temoa energy system model is written with Pyomo, which is in turn written in Python. Consequently, Temoa will run on Linux, Mac, Windows, or any operating system that Pyomo supports.

That said, getting to a functional Coopr installation has historically proven a somewhat difficult task, so we offer two modes of installation:

Simple Installation

The simple installation runs Coopr and Temoa inside of a virtual machine. We have setup a Linux distribution with Coopr, a solver, and Temoa preinstalled. Follow these steps to run it:

  1. Download and install VirtualBox.
  2. Download and install a Torrent application. We suggest μTorrent or Transmission. (Note that Ubuntu may already come with Transmission.)
  3. Download TemoaBox.ova (a "magnet" link that requires a Torrent application).

Why not your normal download process? Or "What that heck is a Torrent?!"

The BitTorrent protocol is a peer-to-peer method to distribute large files over the net. In a nutshell, large files are broken up into small pieces, and downloaders trade these pieces amongst themselves until they have the entire file. In this way, downloaders share their otherwise unused (upload) bandwidth, which means every user benefits from higher speeds when downloading large files. Kinda neat, huh?

Advanced Installation

If you know your way around the computer, and how to deal with intermittent network or installation issues, you may appreciate avoiding the overhead of TemoaBox, and instead installing the relevant tools yourself. If you encounter difficulty with this avenue, you may ask for help on our Forum. However, note that we offer minimal support in this area, and so you may have to ask for help elsewhere. Here are the pieces you will need to install/plumb up:

  1. Python v2.7. Temoa requires at least v2.7, and Coopr does not yet support Python 3+.
  2. COOPR. Temoa employs APIs from the COmmon Optimization Python Repository (Coopr); mainly Pyomo and PySP.
  3. A linear optimization solver. We recommend GLPK as it is GPL software and gratis, but any solver for which Coopr has an interface will do.
  4. The Temoa model, temoa.py. (Or, if you'd like to closely follow development, clone the repository.)
  5. (Optional) Test your installation by running some example data sets.

We have tested the installation process on multiple machines, but unexpected conditions inevitably come up. Please use the Forum to ask for help.
