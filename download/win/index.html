Installing Temoa on Windows

Installing Temoa on Windows requires you to install Python v2.7 and SetupTools. Then, the Temoa Installer for Windows will ensure that you have Coopr and the GLPK solver, and will augment your system path as appropriate. (Note that though the Temoa Installer installs GLPK as a baseline solver, you are welcome to use any solver compatible with Coopr.)

Installation Steps

  1. Install Python v2.7. Coopr does not yet support Python v3+, and Temoa requires at least v2.7.
  2. Install SetupTools. SetupTools is a method for installing Python-specific programs and libraries.
  3. Download the Temoa Installer for Windows and double-click to run it. (Note that Python must be installed for this script to run.)
  4. (Optional) Download the example data files for testing.

After installation is complete, you may test your installation by running a simple model.
