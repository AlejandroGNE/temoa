Installing Temoa on Windows

While it is technically possible to utilize Temoa directly on Windows, it has historically proven a difficult task to accomplish. In theory, the same basic steps as for Linux are required, but we do not offer support for Windows:

  1. Install Python, v2.7. Coopr does not yet support python v3+ and Temoa requires at least v2.7.
  2. Install COOPR (for the AML framework).
  3. Download temoa.py.
  4. (optional) Download the example data files for testing.

If you do not have success with the above installation path, we instead invite you to run a Linux distribution in a Virtual Machine. We have a pre-build virtual machine with Linux and Coopr installed, but due to it's size (~2 GiB), we do not currently have permanent storage. If you would like it, please ask on the Forum.
