Installing Temoa on Mac OS X

The Mac OS X platform is currently not as well supported as we might like. Specifically, we do not have a Mac computer with which to test. There is also no precompiled baseline solver (e.g. GLPK) for easy installation. Thus, we currently do not have an install script to automate the process.

There are a couple of options available to Mac OS X users wishing to model with Temoa:

  • Install Linux inside of a VirtualBox. Then, you can follow the Linux based instructions for installation.
  • Install Python and SetupTools independently, then use easy_install to install Coopr (easy_install -U coopr). Finally, compile and install GLPK (or your solver of choice).

To test your efforts, you might try running a simple model.
